🍝 Bella Vista Restaurant Website

A responsive restaurant website built using HTML5 and CSS3 that showcases an authentic Italian dining experience. The project features a modern user interface, menu display, restaurant information, and an online reservation form.

⸻

📌 Project Overview

Bella Vista is a front-end restaurant website designed to provide customers with information about the restaurant, its menu, story, and reservation services.

The website includes multiple sections such as:

* Home Page
* Menu Display
* About Us
* Contact Information
* Reservation Form
* Footer Navigation

⸻

✨ Features

🏠 Home Section

* Attractive hero banner
* Restaurant introduction
* Call-to-action buttons
* Smooth navigation links

🍽️ Menu Section

* Displays featured menu items
* Item descriptions and pricing
* Organized menu categories

📖 About Us Section

* Restaurant history and background
* Information about ingredients and cooking philosophy
* Restaurant showcase image

📞 Contact Section

* Address details
* Phone number
* Email information

📅 Reservation Form

Customers can reserve a table by providing:

* Full Name
* Email Address
* Phone Number
* Reservation Date
* Preferred Time
* Number of Guests
* Special Requests

🔗 Footer Section

* Quick navigation links
* Opening hours
* Social media links
* Copyright information

⸻

🛠️ Technologies Used

* HTML5
* CSS3
* Responsive Web Design
* Semantic HTML Structure
